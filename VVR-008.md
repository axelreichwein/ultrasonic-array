# VVR-008 Static Obstacle Stop Test

## Requirement

The AN ultrasonic near-field software shall stop the robot within **200 mm**
for a static frontal obstacle, consistently across **20 trials**.

## Software configuration under test

- Node: `lidar_proc`
- Sensor count: 4 transducers
- Processing rate: 10 Hz
- Emergency stop threshold: 150 mm
- Safety compliance margin: 50 mm

## Pass criteria

For each of 20 trials:

- `emergency_stop` must assert when obstacle distance is <= 150 mm
- `compliance_distance_mm` (= nearest obstacle + 50 mm margin) must be <= 200 mm

## Automated implementation

Automated in:

- `ultrasonic_array/test/test_vvr_008_static_obstacle_stop.py`

This test executes 20 deterministic trials and fails on first non-compliant run.
